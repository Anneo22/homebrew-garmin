# Anneo22/garmin

Homebrew tap for [garmin-voice-export](https://github.com/Anneo22/garmin-voice-export):
import your Garmin watch's voice notes to your Mac, automatically.

```sh
brew install anneo22/garmin/garmin-voice-export
garmin-voice-setup        # one-time: installs the on-connect agent + menu-bar app
```

Then plug in your watch. Control it with `garmin-voice status | pause | resume | sync`.
